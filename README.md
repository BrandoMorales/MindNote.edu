# React Navbar Project 🚀

Este proyecto contiene una **Navbar responsiva** hecha con **React + TailwindCSS**.

## 🚀 Cómo usarlo

1. Instala las dependencias:
   ```bash
   npm install
   ```

2. Inicia el servidor de desarrollo:
   ```bash
   npm run dev
   ```

3. Abre en tu navegador la URL que aparece en la terminal (ej: http://localhost:5173).

---

✅ Incluye:  
- Navbar responsiva con menú hamburguesa.  
- Colores personalizados en Tailwind.  
- Ejemplo de secciones (Inicio, Servicios, Contacto).
